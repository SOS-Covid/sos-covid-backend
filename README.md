## Servidor de desenvolvimento 🚀

Clonando o projeto:

```
git clone https://github.com/SOS-Covid/sos-covid-backend.git
```

Navegando até a pasta raiz do projeto:

```
cd sos-covid-backend
```

Baixando as dependências do projeto

```bash
npm i ou npm install
```

Rodando o projeto

```bash
Renomear o arquivo .env.example para .env
```

```bash
npm run dev ou npm start
```

Visite [http://localhost:8080](http://localhost:8080) para chamar as rotas.
